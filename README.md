## 💼 Personal Portfolio Website

## 📌 Overview

This is my **personal portfolio website** designed to showcase my projects, technical skills, and development experience.

## 🚀 Features

* 👩‍💻 About Me section
* 🛠️ Skills and technologies
* 📂 Projects showcase
* 📞 Contact information
* 📱 Fully responsive design


## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Projects Included

### 🍽️ Restaurant Website

A responsive restaurant website with modern UI design, showcasing menu, services, and contact details using HTML, CSS, and JavaScript.

### 🍬 Sweet Shop Website

A visually appealing website for a sweet shop displaying products, categories, and user-friendly navigation.


### 🌸 Flower Shop Website

An elegant flower store website designed with attractive layouts and smooth user experience.


### 🎵 Music Website

A web-based music platform featuring song listings and interactive UI for browsing and playing music.

### 🩸 Blood Flow Monitoring (PPG)

A simulation-based project demonstrating non-invasive blood flow monitoring using PPG signals with interactive waveform visualization.
